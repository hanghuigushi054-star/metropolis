## Sky Metropolis (スカイメトロポリス)
リアクト、@react-three/fiber、@react-three/drei（Three.js）を使用して構築された、フルスタックの3Dアイソメトリック・シティービルダー（街づくりゲーム）です。さらに、Gemini API をゲームのシステムに統合し、「AIアドバイザー」として機能するユニークな仕組みが用意されています。
## 🌟 主な機能・実装済みのシステム
1.**3D地形生成とマップシステム:**
シプレックスノイズを利用して、海（水）、砂浜、草原、森、岩場など高低差とバリエーションのある島（地形）を自動生成します。
アイソメトリック（等角投影）カメラによる視点と、マップのパン/ズームなどの直感的な操作が実装されています。
所有地と未所有地の概念があり、未所有の土地は資金を使って「土地を購入」することで拡大していくことができます。
2.**建設システム:**
道路、住宅、商業施設、工業施設、公園などの建築物を配置したり、破壊（撤去）したりできます。
建設時にはポップアップするように建物が建つ、「スプリングアニメーション」が組み込まれており、リッチな視覚的フィードバックがあります。
プレビュー機能により、設置する前や土地を購入する前に状況を確認できるようになっています。
3.**リソース管理とプログレッション:**
資金、人口、経過日数、プレイヤーレベルの管理システムがあります。
施設の建設には資金が必要で、ゲームティック（ターン/日数）が経過するごとに都市の状況に合わせて人口や資金の増減が行われます。
人口が増えるとレベルが上がり、より上位の施設がアンロックされるなどの進行要素が存在します。
4.**AI インテグレーション (Gemini API実装):**
  * AI 目標システム: AIが現在の都市の様子（資金・人口・日数など）を評価し、次に目指すべき「クエスト（ミッション）」を与えてくれます。達成すると報酬がもらえます。
  * ニュース・イベント通知: 定期的にAIが街の中で起きている出来事やイベントをテキストとして生成し、画面上にニュースフィードとして表示してくれます。
※AI APIのクォータ（利用制限）に達した場合は、自動的に「サンドボックスモード」に移行し、AIイベント無しで建築と拡張を楽しめるようになっています。
5. **UI/HUD:**
都市の各種ステータスバー、建設ツールの選択パレット、AIが提案する目標の表示枠、最新情報が流れるニュースフィードなどがTailwind CSSとReactコンポーネントで綺麗にレイアウトされています。
